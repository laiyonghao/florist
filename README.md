# Florist

种花家，基于Flask的一个web dev实验，准备用来写一本关于Flask项目工程化开发的书。

## 目标

基于 Flask+MongoDB 打造一个快速开发方案，达到开发官网、小店、博客只需要写几十行代码就可以，开发复杂网站六成功能开箱即用的程度。

## 使用的库

仅列出项目内直接 `import` 的

* Flask
* Flask-Mongoengine
  * Mongoengine
* Flask-Admin
* Flask-Security-Too
* Flask-WTForms
  * WTForms
* Flask-CKEditor
* Flask-thumbnails
* flaskfilemanager

## 功能模块

1. meterial: 素材管理
2. facet: 片面模式
3. event: 事件模式
4. cms: 内容管理
5. shop: 店铺功能
6. pay: 支付功能
7. page: 低代码页面开发
8. config: 配置
