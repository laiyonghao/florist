# florist

种花人，基于flask的一个web dev实验。

## 目标

基于 Flask+MongoDB 打造一个快速开发方案，达到开发官网、小店、博客只需要写几十行代码就可以，开发复杂网站六成功能开箱即用的程度。

## 使用的库

仅限项目内直接 `import` 的

* Flask
* Mongoengine

* Flask-Mongoengine
* Flask-Admin

* Flask-Security
* Flask-Security-Too

* WTForms
* Flask-WTForms
